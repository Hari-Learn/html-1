2. Why do we need Active Directory?

Without Active Directory:

  • Every computer needs its own username and password
  • IT team must configure each system manually
  • No central control
  • More mistakes and security issues

With Active Directory:

  • One username & password works everywhere
  • Central control for IT admins
  • Better security
  • Easy management
