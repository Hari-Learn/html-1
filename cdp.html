Switching Essentials

PortFast • CDP • LLDP — what matters in real networks

What is PortFast?

PortFast allows an access port to skip Spanning Tree states and go immediately into forwarding.

Why it exists

  • Eliminates 30–50s STP delay
  • Faster host boot & DHCP
  • Better user experience

When to use

  • PCs, printers
  • IP phones
  • Access ports only

Risks

  • Can create loops
  • Never use on trunk links

Common commands

interface Gig1/0/10
spanning-tree portfast
spanning-tree bpduguard enable

What is CDP?

Cisco Discovery Protocol is a Cisco-only Layer 2 protocol that advertises device information to directly connected neighbors.

What it shows

  • Device hostname
  • IP address
  • Connected interface
  • Platform & OS

Security note

CDP leaks network details. Disable it on external or untrusted ports.

Useful commands

show cdp neighbors
show cdp neighbors detail
no cdp enable

What is LLDP?

LLDP is an open-standard (IEEE 802.1AB) Layer 2 discovery protocol used in multi-vendor environments.

Why engineers use it

  • Vendor-neutral
  • Topology visibility
  • Works with phones/APs

LLDP-MED

  • Voice VLAN info
  • QoS parameters
  • IP phone support

Reality check

Often disabled by default. You must enable transmit and receive.

Commands

lldp run
show lldp neighbors
show lldp neighbors detail
