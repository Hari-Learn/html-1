Layer 2 Switch Communication & Protection (Without VLAN)

1️⃣ Default Setup – All Systems Can Communicate

How It Works

In a normal Layer 2 switch, all devices are in the same broadcast domain. Every system can communicate with each other directly.

Example

  • PC1 → Fa0/1
  • PC2 → Fa0/2
  • PC3 → Fa0/3
  • All systems can ping each other

Configuration

Switch(config)# interface range fa0/1 - 3
Switch(config-if)# switchport mode access

2️⃣ Protected Port – Block Communication Between Systems

How It Works

Protected ports prevent systems connected to those ports from communicating with each other.

  • PC1 ❌ PC2
  • PC2 ❌ PC3
  • All PCs ✔ Router

Configuration

Switch(config)# interface range fa0/1 - 3
Switch(config-if)# switchport protected

Do NOT configure the uplink/router port as protected.

✔ Recommended Method Without VLAN

Use Protected Port / Port Isolation to prevent systems from communicating with each other while still allowing router or internet access.
