Layer 2 Switch Communication & Protection (Without VLAN)

1️⃣ Default Setup – All Systems Can Communicate

How It Works

In a normal Layer 2 switch setup, all connected devices belong to the same broadcast domain. This means every system can communicate directly with every other system.

Example Scenario

  • PC1 connected to Fa0/1
  • PC2 connected to Fa0/2
  • PC3 connected to Fa0/3
  • All PCs can ping each other

Configuration

Switch(config)# interface range fa0/1 - 3
Switch(config-if)# switchport mode access

No additional configuration required. Communication is open.

2️⃣ Protected Port – Block Communication Between Systems

What Protected Port Does

Protected ports prevent devices connected to those ports from communicating with each other. However, they can still communicate with the uplink or router port.

  • PC1 cannot talk to PC2
  • PC2 cannot talk to PC3
  • All PCs can still access router/internet

Configuration Example

Switch(config)# interface range fa0/1 - 3
Switch(config-if)# switchport protected

Ensure uplink/router port is NOT configured as protected.

✔ Recommended Method Without VLAN: Use Protected Port
