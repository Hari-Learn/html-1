Layer 2 Switch Communication & Protection

Without Using VLAN

Default Setup – All Systems Communicate

How It Works

All connected PCs belong to the same broadcast domain. Every system can directly communicate with others.

Example

  • PC1 → Fa0/1
  • PC2 → Fa0/2
  • PC3 → Fa0/3
  • ✔ All can ping each other

Configuration

Switch(config)# interface range fa0/1 - 3
Switch(config-if)# switchport mode access

Protected Port – Isolation Method

How It Works

  • PC1 ❌ PC2
  • PC2 ❌ PC3
  • ✔ PCs can access Router

Configuration

Switch(config)# interface range fa0/1 - 3
Switch(config-if)# switchport protected

Do not configure the uplink/router port as protected.

Simple Network Diagram 

      PC1      PC2      PC3
        |        |        |
        --------------------
               SWITCH
                 |
               ROUTER

In Protected Mode: PCs cannot communicate with each other but can reach the router.
