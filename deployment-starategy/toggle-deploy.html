Feature Toggle (Feature Flags)

Imagine your app has a new feature, but you don’t want everyone to see it yet.

So you add a switch (like a light switch) to turn the feature ON or OFF anytime.

How it works:

  • Feature is deployed but hidden
  • You can turn it ON for some users
  • You can turn it OFF instantly if something breaks

This gives you full control without redeploying the app.

Simple example:

Like adding a new game level but unlocking it only for a few players first.
