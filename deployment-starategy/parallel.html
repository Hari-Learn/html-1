Parallel Run

Imagine you built a new calculator, but you don’t trust it fully yet.

So you use both the old calculator and the new one at the same time.

How it works:

  • Run old system and new system together
  • Compare results from both
  • Make sure they match

Once you're sure the new one works perfectly, you stop using the old one.

Simple example:

Like double-checking your homework with two different methods.
