Technician Command Cheat Sheet

🖥️ When PC is Slow or Freezing

  • %temp% → Clear temporary files
  • temp → Delete temp files
  • prefetch → Clean prefetch files
  • cleanmgr → Disk cleanup tool
  • taskmgr → Open Task Manager
  • services.msc → Manage services
  • sfc /scannow → Scan system files
  • chkdsk /f → Fix disk errors

🌐 When Internet is Not Working

  • ping → Test connectivity
  • ipconfig → Show IP info
  • ipconfig /release → Release IP
  • ipconfig /renew → Renew IP
  • ipconfig /flushdns → Flush DNS cache
  • nslookup → Check DNS resolution
  • tracert → Trace network route
  • netsh int ip reset → Reset network stack
Quick reference for IT technicians 🚀
