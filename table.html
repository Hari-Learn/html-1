Docker Containers
Docker Containers Cheat Sheet
Category Commands Details
General
docker --d Start the docker daemon
docker --help Get help with Docker.Can also use -help on subcommands
Images
docker build -t image-name Build an image from Docker file
docker build -t image-name --no-cache Build an image from Docker file without the cache
docker images List lcoal images
docker rmi image-name Delete an images
docker image prune Remove all unused images
Container Registry
docker login -u user-name Login to Docker Hub
docker push username/imagename:tag Publish an image to Docker hub
docker pull image-name:tag Pull an image from Docker hub
docker tag imageName newName:version Rename an existing tag name
Status
docker stats --all Docker stats of all container
docker top container-name or container id Display the running processes of a container
docker history imageName or imageId Shows history of docker image
Containers
docker run --name containerName imageName Create and run a container from an image, with a custom name
docker run -d imageName Run a container in a background
docker run -p hostPort:containerPort imageName Run a container and publish a container ports to the host.
docker start/stop containerName or containerId Start or Stop a container
docker rm containerName Remove a stopped container
docker exex -it containerName sh Open or attach a shell inside a running container
docker kill containerName Kill/stop a running a container
docker logs -f containerName Fetch and follow the logs of the container
docker inspect containerName/containerId To inspect a running container
docker ps To list currently running container
docker ps --all List all container running and stopped
docker container stats View resource usage stats
Networking
docker network ls List all the networks
docker network create --driver driverName bridgeName Create a network`
docker connect network networkName containerName Connect a running container to a network`
docker network disconnect networkName containerName Disconnect a container from a network`
docker network rm networkName Remove a network
Docker Compose
docker compose up Create and start containers
docker compose down Stop and remove containers network
docker compose logs View output from containers
docker compose events Receive real time events from containers
docker compose ps List containers launched as part of compose
