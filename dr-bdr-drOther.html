OSPF Router ID & DR / BDR Concepts

Router ID • DR • BDR • DROTHER • Multicast 224.0.0.6

What is Router ID (RID)?

Router ID is a 32-bit unique identifier used by OSPF to identify each router in the OSPF domain. It looks like an IP address but does not have to be routable.

OSPF uses Router ID during:

  • Neighbor formation
  • DR / BDR election
  • LSA identification

Conditions to Select Router ID

OSPF selects Router ID in the following priority order:

  1. Manually configured Router ID
    (router-id x.x.x.x) – highest priority
  2. Highest IP address on Loopback interface
  3. Highest IP address on active physical interface

⚠️ Router ID is selected only once at OSPF startup. Changes require OSPF reset.

What is DR (Designated Router)?

DR is the central router on broadcast and NBMA networks responsible for reducing OSPF overhead.

All routers form full adjacency with the DR instead of each other.

DR sends updates using multicast 224.0.0.5.

What is BDR (Backup Designated Router)?

BDR is the standby router that takes over when the DR fails.

BDR maintains full adjacency and stays synchronized with the DR.

✅ Ensures high availability and fast convergence.

What is DROTHER?

DROTHER refers to routers that are neither DR nor BDR.

DROTHER routers:

  • Form full adjacency only with DR and BDR
  • Stay in 2-Way state with other DROTHER routers

What is Self Election in OSPF?

OSPF uses self-election for DR/BDR selection.

Each router advertises its:

  • OSPF Priority (default = 1)
  • Router ID

The router with highest priority becomes DR. If priorities tie, highest Router ID wins.

What is One-Time Election?

DR/BDR election happens only once when OSPF neighbors come up.

If a new router with higher priority joins later, it will NOT replace the existing DR.

⚠️ DR change occurs only if DR fails or OSPF process restarts.

What is 224.0.0.6 and Why Is It Used?

224.0.0.6 is the multicast address for All OSPF Designated Routers.

DROTHER routers send LSAs to 224.0.0.6 so that:

  • Only DR and BDR process the update
  • Network flooding is reduced

DR then floods updates to all routers using 224.0.0.5.
