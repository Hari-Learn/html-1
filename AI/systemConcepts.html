AI System Design

🔗

A2A Communication

A2A is like **AI talking to other AI**. Imagine teammates passing messages to solve a problem together.

Each agent shares useful info so the whole system can make better decisions.

🧠

Memory & State

This is like **AI remembering things**. Short-term memory is what it just saw, and long-term memory is what it learned earlier.

Good memory helps AI stay consistent across conversations.

📦

Context Window

Think of this as **how much the AI can "see" at once**.

A bigger context window means the AI can handle longer conversations, more details, and solve more complex problems without forgetting.
