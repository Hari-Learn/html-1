Overlay vs Underlay Network Virtualization

Understanding Virtual Topology and Physical Connectivity

Network Virtualization

Network virtualization is the process of creating multiple virtual networks on top of a shared physical network infrastructure. It allows organizations to logically divide a network so that multiple users or applications can run independently without affecting each other.

Technologies such as VXLAN, GRE, and SDN controllers are commonly used to implement network virtualization in modern data centers and cloud environments.

Underlay Network

The underlay network refers to the physical network infrastructure that connects devices. It includes routers, switches, cables, and IP routing that provide the basic connectivity between all devices in the network.

  • Consists of physical routers and switches
  • Provides IP connectivity
  • Handles packet forwarding between devices
  • Usually built using traditional routing protocols

Overlay Network

The overlay network is a virtual network created on top of the underlay network. It uses tunneling protocols to connect virtual machines or containers across the physical network.

  • Built on top of the underlay network
  • Creates logical connections between endpoints
  • Uses tunneling protocols such as VXLAN
  • Provides network isolation and flexibility

Virtual Topology

Virtual topology refers to the logical layout of devices within the overlay network. It defines how virtual machines, containers, or applications communicate with each other regardless of their physical location in the data center.

For example, two virtual machines located in different physical servers may appear directly connected in a virtual network.

Physical Connectivity

Physical connectivity represents the real-world connections between networking devices using cables, switches, and routers. This is part of the underlay network and ensures that data packets can travel across the infrastructure.

  • Uses Ethernet cables and fiber links
  • Connects switches, routers, and servers
  • Provides the foundation for overlay networks

Overlay vs Underlay

Feature Underlay Network Overlay Network
Type Physical infrastructure Virtual network
