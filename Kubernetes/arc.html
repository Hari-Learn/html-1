Architecture Explained

Master Nodes + Worker Nodes

Kubernetes Cluster

A cluster is a group of computers working together.

Master Node

  • 🧠 Brain of Kubernetes
  • 📋 Makes decisions
  • 📡 Controls cluster

Worker Node

  • ⚙️ Runs applications
  • 📦 Hosts containers
  • 🚀 Handles workloads

Pods

Pods are the smallest unit in Kubernetes. They contain containers.

Services

Services help apps communicate with each other.

Important Components

API Server

Front door of Kubernetes.

Scheduler

Assigns workloads.

etcd

Stores cluster data.

Controller

Keeps system healthy.
