API Communication (Page 2)

📊 Status Codes

These are numbers that tell what happened.

200 = success, 404 = not found, 500 = error.

🔐 Authentication

This checks who you are.

Like logging into your account.

⚙️ Parameters

Extra details you send with a request.

Like choosing size or color.

📑 Headers

Small extra information sent with a request.

Like writing details on an envelope.

✅ Authorization

This checks what you are allowed to do.

Like permission to enter a room.
