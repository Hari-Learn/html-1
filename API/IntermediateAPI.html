💬 API Communication (Page 2)

📊 Status Codes

These are like reply messages from the server.

They tell you what happened:

  • ✅ 200 → Everything worked!
  • 🔍 404 → Couldn't find it
  • 💥 500 → Server had a problem

🔐 Authentication (Who are you?)

This checks your identity.

Like logging into your account with a username and password.

👉 "Are you really you?"

⚙️ Parameters (Extra choices)

These are extra details you send.

Like ordering a pizza 🍕:

  • Size: Large
  • Toppings: Cheese 🧀

👉 These choices = parameters

📑 Headers

Headers are small hidden details sent with your request.

Like writing info on an envelope ✉️:

  • Who sent it
  • What type of data it is

✅ Authorization (What can you do?)

This checks your permissions.

Like entering a room 🚪:

  • Guest → can only look 👀
  • Admin → can change things ✏️

👉 "Are you allowed to do this?"

🎯 First prove who you are → then check what you can do!
