🚀 API Advanced Concepts (Page 3)

⏱️ Rate Limiting

This controls how many times you can ask the server in a short time.

Like a game that says: "You can only try 5 times per minute" 🎮

👉 It stops overload and keeps servers safe

🔑 API Key

This is a secret ID used to access an API.

Like a special ticket 🎟️ that lets only you enter.

👉 Without it, the API says "No entry!"

🧾 JSON (Data Language)

JSON is a simple way to send data between apps.

It looks like pairs: 

{
  "name": "John",
  "age": 10
}

👉 Think of it like a labeled box 📦

🔔 Webhook

A webhook is like an automatic messenger.

It sends you a message 📩 when something happens.

Example: “Your order is shipped 🚚”

🔐 OAuth (Login with others)

OAuth lets you log in using another account.

Like using Google or Facebook instead of creating a new login.

👉 You don’t share your password directly

🎯 APIs are just smart messengers that follow rules!
