API Advanced Concepts (Page 3)

⏱️ Rate Limiting

This limits how many times you can ask the server.

Like "only 5 tries per minute".

🔑 API Key

A secret code to use the API.

Like a password for apps.

🧾 JSON

A simple format to send data.

It looks like key-value pairs.

🔔 Webhook

A webhook sends data automatically when something happens.

Like getting a notification.

🔐 OAuth

Lets you log in using another account.

Like "Login with Google".
