API Basics (Page 1)

🔗 What is API?

An API is like a waiter in a restaurant.

You tell the waiter what you want, and it brings it from the kitchen.

📍 Endpoint

An endpoint is like a specific door to get something.

Each door gives you different information.

📬 HTTP Methods

These are actions like GET (read), POST (send), PUT (update), DELETE (remove).

Like telling what you want to do.

📤 Request

A request is when you ask the server for something.

Like placing an order.

📥 Response

A response is what the server sends back.

Like receiving your food.
