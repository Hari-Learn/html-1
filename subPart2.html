Subnetting Tutorial – Part 2

Computers understand IP addresses in binary (0s and 1s).

Binary Basics

Each octet has 8 bits:

128 64 32 16 8 4 2 1

Example: 192 = 128 + 64

Subnet Mask

Subnet mask tells which part is network and which part is host.

/24 = 255.255.255.0

/26 = 255.255.255.192
