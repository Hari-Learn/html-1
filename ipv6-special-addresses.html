IPv6 Special Addresses

Quick Revision Cards for CCNA Aspirants

::1

Loopback Address

Used by a device to refer to itself. Equivalent to IPv4 127.0.0.1.

📝 Exam Tip: Used for local testing.

::

Unspecified Address

Represents the absence of an address. Used during device boot before IP is assigned.

📝 Exam Tip: Means "no address yet".

FE80::/10

Link-Local

Automatically assigned. Used for communication within the same local network.

📝 Exam Tip: Routers do NOT forward these.

FF00::/8

Multicast

Used to send packets to multiple destinations at once.

📝 Exam Tip: Replaces broadcast in IPv6.

2000::/3

Global Unicast

Publicly routable IPv6 addresses on the internet. Equivalent to public IPv4.

📝 Exam Tip: Most common type in production networks.

Quick Memory Trick

  • ::1 → 1 device (yourself)
  • :: → Empty / Nothing
  • FE80 → "FEels local"
  • FF → "Friends Forever" (group)
  • 2000 → Internet Public Range
