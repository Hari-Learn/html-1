Network Devices Explained

Hub vs Switch vs Router (Simple & Visual)

📡

Hub

A hub is like a **loudspeaker** — it shouts data to everyone.

  • • Works at Layer 1 (Physical)
  • • Sends data to all devices
  • • No intelligence
  • • More traffic & collisions
  • • Least secure
🔀

Switch

A switch is like a **smart delivery person** — sends data only where needed.

  • • Works at Layer 2 (Data Link)
  • • Uses MAC addresses
  • • Sends data to correct device
  • • Faster & efficient
  • • More secure than hub
🌐

Router

A router is like a **GPS navigator** — finds the best path for data.

  • • Works at Layer 3 (Network)
  • • Uses IP addresses
  • • Connects different networks
  • • Routes data efficiently
  • • Adds security (firewall, NAT)

How Data Flows

Hub (Broadcast) ➡️ Switch (Smart Send) ➡️ Router (Path Finder)
