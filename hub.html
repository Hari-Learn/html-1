Behavior of Network Devices

Hub, Switch, and Router

Hub

A Hub is a basic networking device that connects multiple computers in a network.

  • Works at the Physical Layer (Layer 1).
  • Sends data to all connected devices.
  • Does not identify destination devices.
  • Causes more network traffic and collisions.
  • Less secure and slower.

Switch

A Switch is a smarter device used to connect devices within a local network.

  • Works at the Data Link Layer (Layer 2).
  • Uses MAC addresses to send data.
  • Sends data only to the intended device.
  • Reduces collisions and improves speed.
  • More secure than a hub.

Router

A Router connects different networks and directs data between them.

  • Works at the Network Layer (Layer 3).
  • Uses IP addresses to route data.
  • Connects LAN to the Internet.
  • Selects the best path for data transmission.
  • Provides security features like firewalls.
