Firewall

The firewall is the **first security checkpoint** inside your network.

It decides what traffic is safe and blocks harmful requests.

🌐

Core Switch

The core switch is like the **main highway** of the network.

It handles large volumes of traffic and connects major parts of the network.

🔀

Distribution Switch

This layer organizes traffic between core and access layers.

Think of it as a **traffic controller** directing data efficiently.

🧩

ToR (Top of Rack)

Located on top of server racks, these switches connect servers directly.

They act like a **local hub** for servers in a rack.

🌍

Web Servers

These are the machines that deliver websites and apps to users.

When you open a site, you're talking to a web server.

🔌

Patch Panels

Patch panels organize network cables neatly.

Think of them like a **plug board** that keeps connections clean and manageable.

🧵

Structured Cabling

This is the organized system of cables running through the building.

It ensures everything is connected reliably and easy to maintain.

🗄️

Server Racks

Server racks hold servers and networking equipment in an organized way.

Like **shelves for computers**, keeping everything tidy and cool.

Simple Internal Flow

Firewall → Core → Distribution → ToR → Servers
