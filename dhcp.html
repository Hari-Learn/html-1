What is DHCP?

DHCP (Dynamic Host Configuration Protocol) is a network protocol that automatically assigns IP addresses and other network settings to devices. This removes the need for manual IP configuration.

DHCP DORA Process

1. Discover

The client broadcasts a DHCP Discover message to find available DHCP servers.

2. Offer

The DHCP server responds with an Offer, providing an available IP address.

3. Request

The client requests the offered IP address from the server.

4. Acknowledge

The server acknowledges and assigns the IP configuration to the client.

APIPA (When DHCP is Unavailable)

If the DHCP server is unavailable or unreachable, the client assigns itself an APIPA (Automatic Private IP Addressing) address.

  • IP range: 169.254.0.0 – 169.254.255.255
  • Allows limited communication within the local network
  • No internet or routed network access
