ARP Packet Flow

Address Resolution Protocol (ARP) is used to find the MAC address associated with an IP address in a local network.

Host A

IP: 192.168.1.10

MAC: AA:AA:AA:AA:AA:AA

Switch

Broadcasts ARP Request

Host B

IP: 192.168.1.20

MAC: BB:BB:BB:BB:BB:BB

Step-by-Step ARP Flow

1. ARP Request (Broadcast)

“Who has IP 192.168.1.20? Tell 192.168.1.10”

2. Switch Forwards

The switch sends the request to all devices.

3. ARP Reply (Unicast)

Host B replies with its MAC address.

4. ARP Table Updated

Host A stores IP–MAC mapping and can now send data.

ARP works only within a local network (LAN)
