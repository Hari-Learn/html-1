Event Delegation (ELI10)

Imagine you are a teacher 👩‍🏫 in a classroom. Instead of telling each student individually what to do, you just give one instruction to the whole class.

In JavaScript, event delegation works the same way. Instead of adding a click event to every single button, you add one event to their parent, and it listens for all of them.

We did NOT add click events to each item. Instead, we added ONE event to the parent list. When you click an item, the parent figures out which one you clicked. 

// One parent event instead of many
document.getElementById("list").addEventListener("click", function(e) {
  if (e.target.tagName === "LI") {
    alert("You clicked " + e.target.innerText);
  }
});

✅ Less code
✅ Faster
✅ Works even for new items added later!
