Security Concepts

Stateful Firewall

Tracks connection states and permits traffic based on session information rather than static rules only.

NAT & PAT

Network Address Translation allows private networks to securely access public networks.

VPN Technologies

Supports Site-to-Site IPsec VPN, Remote Access VPN, and SSL VPN.

NGFW Features

Application awareness, intrusion prevention, URL filtering, and malware protection.

Cisco ASA Firewall

Overview

Cisco ASA (Adaptive Security Appliance) is a stateful firewall platform that evolved from Cisco PIX, offering enhanced performance, modular services, and enterprise-grade security.

Key Capabilities

  • Stateful packet inspection
  • Access Control Lists (ACLs)
  • NAT and PAT
  • High availability (Active/Standby)
  • IPsec & SSL VPN

Limitations

Traditional ASA lacks deep application visibility and advanced threat analytics compared to modern NGFW solutions.

FTD (Firepower Threat Defense)

FTD merges ASA firewall functionality and Firepower services into a single image managed centrally via FMC.

ASA vs FTD

  • ASA: Traditional firewall + VPN
  • FTD: Full NGFW with threat intelligence
