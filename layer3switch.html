Layer 3 Switch

Routing and Switching Combined

What is a Layer 3 Switch?

A Layer 3 Switch is a network device that performs both switching (Layer 2) and routing (Layer 3) functions.

Unlike a normal switch, it can make routing decisions using IP addresses, similar to a router, but at much faster speeds.

Routing Inside a Layer 3 Switch

Routing inside a Layer 3 switch is done using SVIs (Switched Virtual Interfaces).

Each VLAN is assigned an IP address, and the switch routes traffic between VLANs internally without sending it to an external router.

Why Did Layer 3 Switches Come?

Traditional networks used routers for inter-VLAN routing, which caused performance bottlenecks.

  • Routers were slower compared to switches
  • Extra cabling and complexity
  • High traffic caused delays

Layer 3 switches were introduced to overcome these limitations.

What Problems Does It Solve?

  • Eliminates router-on-a-stick bottleneck
  • Faster inter-VLAN routing
  • Reduced network latency
  • Simpler network design

Why Do We Need a Layer 3 Switch?

  • High-speed routing within LAN
  • Supports large enterprise networks
  • Better scalability
  • Cost-effective compared to multiple routers

How to Configure a Layer 3 Switch (Basic Example)

Example: Inter-VLAN routing using SVIs 

enable
configure terminal

! Enable routing
ip routing

! Create VLANs
vlan 10
vlan 20

! Configure SVI for VLAN 10
interface vlan 10
 ip address 192.168.10.1 255.255.255.0
 no shutdown

! Configure SVI for VLAN 20
interface vlan 20
 ip address 192.168.20.1 255.255.255.0
 no shutdown

After this, devices in VLAN 10 and VLAN 20 can communicate without a router.
