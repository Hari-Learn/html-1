Advanced STP Features

Spanning Tree protection mechanisms beyond basic STP

Root Guard

Prevents an access or distribution switch from becoming the root bridge by blocking ports that receive superior BPDUs.

Loop Guard

Protects the network from loops caused by unidirectional link failures when BPDUs stop arriving.

BPDU Guard

Immediately places a PortFast-enabled port into err-disabled state when a BPDU is received.

BPDU Filter

Stops BPDUs from being sent or received on a port. Should be used carefully as it can introduce loops.

Why These Matter (CCNA)

These advanced STP features are considered rare but high-value CCNA topics. They focus on protecting the network from misconfigurations, rogue switches, and Layer 2 loops.
