Subnetting Tutorial – Part 1

Subnetting is the process of dividing a large network into smaller networks called subnets.

Why Subnetting?

  • Reduces network congestion
  • Improves security
  • Efficient use of IP addresses

IP Address Structure

An IPv4 address has two parts:

  • Network Portion – Identifies the network
  • Host Portion – Identifies devices in the network

CIDR Notation

CIDR uses a slash notation like /24.
Example: 192.168.1.0/24
