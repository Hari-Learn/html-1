SD-WAN Basics

Software-Defined Wide Area Networking

What is SD-WAN?

SD-WAN (Software-Defined WAN) is a modern approach to managing wide-area networks using centralized control and automation. It separates networking hardware from its control mechanism to improve flexibility and efficiency.

1️⃣ Separation of Control & Data Plane

The control plane makes decisions about where traffic is sent, while the data plane actually forwards the traffic. SD-WAN separates these functions for better scalability and management.

2️⃣ Overlay vs Underlay Networks

The underlay is the physical network (Internet, MPLS, LTE). The overlay is the virtual network built on top, used to securely route traffic across multiple transport types.

3️⃣ Cisco SD-WAN Components

  • vManage: Centralized management dashboard
  • vBond: Orchestrates device authentication
  • vSmart: Central control plane

4️⃣ Why Enterprises Replace MPLS

  • Lower cost compared to MPLS
  • Better cloud connectivity
  • Centralized automation
  • Improved performance & redundancy
