BGP Cheat Sheet (CCNA)

One-page quick revision for the CCNA exam

BGP Basics

  • Full form: Border Gateway Protocol
  • Type: Path Vector
  • Used for: Inter-AS routing
  • Internet routing protocol

Transport

  • Protocol: TCP
  • Port: 179
  • Reliable: Yes
  • No periodic updates

Autonomous System

  • Collection of networks
  • Single admin control
  • Identified by ASN
  • Public & Private ASNs

eBGP vs iBGP

  • eBGP: Between different AS
  • iBGP: Same AS
  • iBGP needs full mesh
  • TTL: eBGP 1 hop (default)

BGP Attributes

  • Weight (Cisco only)
  • Local Preference
  • AS Path
  • Origin
  • Next Hop

Path Selection (Top)

  1. Highest Weight
  2. Highest Local Preference
  3. Shortest AS Path
  4. Lowest Origin Type

Updates

  • Incremental updates
  • No periodic full updates
  • Uses keepalives
  • Slower convergence

Characteristics

  • Classless protocol
  • Highly scalable
  • Policy-based routing
  • Not an IGP

CCNA Must-Remember

  • ✔ BGP ≠ OSPF
  • ✔ TCP Port 179
  • ✔ Path Vector protocol
  • ✔ Uses attributes, not metrics
