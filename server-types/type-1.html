Types of Servers (Page 1)

📁 File Server

A file server is like a big cupboard where all your files are stored.

People in a school or office can open it and take the files they need.

🌐 Web Server

A web server shows websites to you.

When you type a website name, it sends the page to your browser.

🗄️ Database Server

This server stores lots of data like names, marks, or products.

Apps ask it questions and it gives the correct data back.
