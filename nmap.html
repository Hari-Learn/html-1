Nmap Quick Guide

What is Nmap?

Nmap (Network Mapper) is a free and open-source tool for network discovery and security auditing.

Common Flags

-sV
Version detection. Helps find vulnerabilities in specific software versions and is useful for troubleshooting.

-A
Aggressive scan. Enables OS detection, version detection, script scanning, and traceroute.

-O
OS detection only. Tries to determine the target operating system.

-sT
TCP connect scan (full TCP handshake). Less stealthy but works when SYN scan is not possible.

-sS
Stealth SYN scan. Sends SYN packets and waits for responses without completing the handshake. Less detectable.
