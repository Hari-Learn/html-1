Core Concepts

Exploit: Code that triggers a vulnerability to gain access.

Payload: Code executed after exploitation (e.g., shell).

  • Exploit Modules
  • Payload Modules
  • Auxiliary/Post Modules

Think of: Metasploit = Exploit + Payload + Handler.

Typical Workflow

  1. Recon: Use nmap to find live hosts and open ports.
  2. Search: In Metasploit: search vsftpd
  3. Set Options:
    • set RHOST 192.168.56.101
    • set LHOST 192.168.56.1
  4. Exploit: Run the module: exploit
  5. Post: Dump hashes, escalate privileges, pivot if needed.

Note: Use Metasploit only in test labs or with explicit authorization.
