Burp Suite Basics - Explain Like I'm 10

Your first step into ethical hacking and web security tools

🕵️ What is Burp Suite?

Burp Suite is a tool that helps security testers (ethical hackers) find problems in websites, like broken locks or hidden doors. It's like using a magnifying glass to inspect how websites talk to your browser.

🔗 Proxy

Acts like a middleman between your browser and the website. It lets you see and change what data is being sent and received.

  • Capture requests
  • Edit before sending
  • Pause and inspect traffic

📡 Repeater

Lets you resend the same request over and over with small changes to see how the website reacts.

  • Manual testing
  • Try different inputs
  • Check responses

🧠 Scanner (Pro only)

Automatically checks for common website issues like insecure login pages, broken access controls, etc.

📁 Intruder

Tries lots of inputs very fast, like guessing passwords or testing forms with many values.

🧪 Decoder

Helps you turn weird-looking data (like %20 or Base64) into something readable.

🚦 Why Do Beginners Use Burp Suite?

  • To learn how websites and browsers communicate
  • To find weak spots in test sites (legally!)
  • To practice ethical hacking safely

🔒 Is This Legal?

Yes, if you have permission! You can only test your own apps or websites you're allowed to test (like PortSwigger’s Web Security Academy).

🚀 Getting Started

  1. Download Burp Suite from portswigger.net/burp
  2. Install it and run it
  3. Set your browser to use the Burp proxy
  4. Visit a test site and inspect the traffic
