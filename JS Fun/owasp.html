Overview

OWASP ZAP is a proxy-based tool that helps security testers intercept, inspect, and manipulate HTTP(S) traffic between a web browser and web applications. It provides automated scanners, passive scanning, scripting, and many integrations suitable for defensive and offensive security learning — always use on authorized targets.

Primary uses
Proxying, spidering, active/passive scanning, fuzzing.
Good for
Beginners learning web app security and proxy workflows.
Platforms
Cross-platform (Windows, macOS, Linux, Docker).

OWASP ZAP docs, official tutorials, and safe lab environments.
