Burp Suite — Cheatsheet (white)

Separate boxes for quick scanning. Use only with permission — be ethical.

Overview

What it is

Burp Suite is a web application security testing platform (PortSwigger) that provides an intercepting proxy and a set of tools to analyze, manipulate, and test web traffic.

Core Tools

  • Proxy — intercept & modify HTTP(S).
  • Target — sitemap & manual inspection.
  • Repeater — craft & resend requests.
  • Intruder — automated fuzzing & payloads.
  • Scanner (Pro) — automated vulnerability scanning.
  • Decoder / Comparer / Sequencer / Extender — helpers & plugins.

Typical Quick Workflow

  1. Set browser proxy → 127.0.0.1:8080 and install Burp CA certificate for HTTPS.
  2. Browse app to populate Target → map endpoints.
  3. Send interesting requests to Repeater / Intruder / Scanner.
  4. Analyze with Decoder, Sequencer, and Comparer.

Shortcuts & Quick Actions

Common shortcuts

  • Ctrl+R → Send selected item to Repeater.
  • Ctrl+I → Send to Intruder.
  • Right-click a request → Send to Repeater / Intruder / Scanner / Comparer.
  • Proxy → HTTP history → search or export selected items.

Quick tips

  • Use "Match and Replace" for repetitive header edits.
  • Turn Intercept OFF while crawling unless you need to modify specific requests.

Intruder Modes

  • Sniper — single position, targeted testing.
  • Clusterbomb — combinatorial multi-position (noisy).
  • Pitchfork — parallel payload sets (aligned).
  • Turbo — fastest execution (use with caution).

Useful BApp Extensions

Install via Extender → BApp Store

  • Logger++ — enhanced logging
  • JSON Beautifier / Pretty Print — easier reading
  • Autorize — authorization testing helper
  • Turbo Intruder (advanced) — high-performance fuzzing

Headers & Response Checks

  • Host, Cookie, Referer, User-Agent — check differences.
  • Compare status codes, body length, timing, and header presence.
  • Use Comparer to diff two responses quickly.

Safety & Legal

Be responsible

Only test systems you own or have explicit permission to test. Unauthorized scanning or exploitation is illegal. Keep logs, report findings clearly, and avoid destructive actions.

Quick Reference

Proxy

Intercept toggle → capture requests; right-click → send to tools.

Repeater

Manual tampering — change parameters, headers, bodies, then resend.

Intruder

Configure positions, select payload sets, choose attack type, run (be cautious).

Scanner (Pro)

Active & Passive checks — use on authorized targets only.
