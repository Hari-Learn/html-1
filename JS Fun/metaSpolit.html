What is Metasploit?

Metasploit Framework is an open‑source platform used for developing, testing, and executing exploit code against remote targets. It's widely used by security professionals for penetration testing and research.

Core concepts

  • Exploit — code that takes advantage of a vulnerability.
  • Payload — code delivered by an exploit (e.g., a reverse shell).
  • Module — organized pieces in Metasploit (exploits, aux, post, payloads).

Typical workflow

  1. Recon & identify a target and vulnerable service.
  2. Select an exploit module and configure options (RHOST, RPORT, etc.).
  3. Choose payload and set listeners (e.g., a reverse TCP handler).
  4. Run the exploit and perform post‑exploitation tasks.

Note: Use Metasploit only in authorized environments (your lab, VMs, or with explicit permission).

Want a tiny example command to try in a controlled lab? Try: msfconsole then search for modules with search service.
