John the Ripper (often abbreviated as john) is a fast and powerful password-cracking tool. It is used by security professionals, ethical hackers, and system administrators to test password strength and recover lost passwords.

‚ùì Why Use It?

Passwords are a common attack vector. John the Ripper helps identify weak or easy-to-guess passwords in systems. It's a valuable tool for:

  • Security audits
  • Penetration testing
  • Recovering forgotten passwords
  • Testing custom password hashes

‚öôÔ∏è How Does It Work?

John the Ripper works by taking hashed passwords (like those from /etc/shadow on Linux) and trying to match them using:

  • Dictionary attacks: Using a wordlist of common passwords
  • Brute force: Trying all possible combinations
  • Hybrid attacks: Modifying dictionary words (like password123)

It supports many hash types: MD5, SHA, DES, bcrypt, and even Windows hashes (LM/NTLM).

  • Audit password policies in organizations
  • Recover lost credentials (with permission)
