What is Maltego?

Maltego is a powerful OSINT (Open Source Intelligence) and link analysis tool that helps you uncover relationships between people, domains, organizations, emails, social media, and more. It creates a dynamic graph that shows how all the data is connected.

How to Use Maltego

  1. Download and install Maltego Community or Professional edition.
  2. Create a new graph and choose a starting point (e.g., domain, IP, email).
  3. Right-click the node and run transforms to fetch more related data.
  4. Expand the graph by exploring connected nodes and patterns.
  5. Use filters, layouts, and notes to refine your investigation.

What Can You Do?

  • Find website ownership and hosting details
  • Discover email addresses tied to accounts
  • Reveal relationships between people and assets
  • Map out attacker infrastructure in cybersecurity
  • Check for breached data and leaked credentials
