Lens

A Lens is a functional programming pattern that provides a simple and reusable way to access and update nested data structures.

Instead of manually navigating deeply nested objects, a Lens focuses on a specific field and allows safe reading and updating.

How It Works

  1. Create a Lens for a specific field.
  2. Use the Lens to read the value.
  3. Use the Lens to update the value.
  4. A new updated object is returned.

Benefits

  • Simplifies nested data access.
  • Supports immutable updates.
  • Reduces repetitive code.
  • Improves readability and maintainability.

Simple Example

A Lens can focus on user.address.city and update the city value without manually copying the entire object structure.
