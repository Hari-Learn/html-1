Applicative Functor

An Applicative Functor is a functional programming pattern that applies wrapped functions to wrapped values.

Unlike a regular Functor, which applies a normal function to a wrapped value, an Applicative Functor can apply a function that is itself wrapped inside a container.

How It Works

  1. Store a function inside a container.
  2. Store a value inside a container.
  3. Apply the wrapped function to the wrapped value.
  4. Return the result in the same type of container.

Benefits

  • Works with multiple wrapped values.
  • Supports functional composition.
  • Maintains consistent container types.
  • Reduces boilerplate code.

Simple Example

A container holds the function x → x + 10 and another container holds the value 5. Applying them produces a container holding 15.

Simple Flow

Box(x → x + 10)
+
Box(5)
↓ apply
Box(15)
