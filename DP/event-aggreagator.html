Event Aggregator Pattern

The Event Aggregator pattern provides a central hub for publishing and subscribing to events.

Component A ─┐
Component B ─┼─► Event Aggregator ─► Subscribers
Component C ─┘
  • Reduces direct dependencies between components.
  • Improves modularity and maintainability.
  • Common in UI frameworks and event-driven systems.

Example: In a dashboard application, multiple widgets publish events to an Event Aggregator, and interested widgets receive only the events they subscribe to.
