Software Testing

Golden Master Testing

Golden Master Testing is a regression testing technique that validates software by comparing the current output with a previously approved "golden master" output. If the outputs match, the application is considered to behave consistently; any differences indicate potential regressions or intentional changes that require review.

Key Concept

Regression testing by comparing outputs. The current execution results are compared against a trusted baseline (Golden Master) to detect unexpected changes automatically.
