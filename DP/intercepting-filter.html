Intercepting Filter Pattern

Category: Enterprise Pattern

Definition:
Apply preprocessing and postprocessing to requests before they reach the target resource.

Purpose

  • Validate incoming requests.
  • Perform authentication and authorization.
  • Log requests and responses.
  • Apply request or response transformations.

Simple Flow

Client → Filter → Target Resource → Response

Example

AuthenticationFilter
        ↓
LoggingFilter
        ↓
RequestHandler
