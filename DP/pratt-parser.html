🧩 Pratt Parser Pattern

A Pratt Parser is an expression parsing technique that uses token precedence (binding power) to parse expressions. It is simple, flexible, and handles operator precedence and associativity without requiring a large grammar.

How It Works

  1. Read the next token.
  2. Parse the left side of the expression.
  3. Compare operator precedence.
  4. Recursively parse the right side if needed.

Pseudo Code

left = parsePrefix();

while (precedence < nextPrecedence()) {
    left = parseInfix(left);
}

return left;

Commonly used in interpreters, compilers, scripting languages, and expression evaluators because it provides an elegant way to parse complex expressions with many operators.
