🎮 Game Loop Pattern

The Game Loop is the core execution cycle of a game. It continuously updates the game state, processes player input, and renders graphics until the game ends.

Basic Flow

  1. Read player input
  2. Update game objects
  3. Detect collisions
  4. Render the scene
  5. Repeat until the game exits

Pseudo Code

while (gameRunning) {
    processInput();
    update();
    render();
}

Used in almost every game engine including Unity, Unreal Engine, Godot, and custom game frameworks.
