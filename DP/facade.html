Facade Design Pattern

Imagine you want to watch a movie at home.

Normally, you must:

  • Turn on the TV
  • Turn on the speaker
  • Switch to HDMI
  • Open Netflix
  • Choose the movie

That is a lot of steps.

Facade Pattern says:

"Hide all those complicated steps behind one simple button."

Simple Definition

A Facade provides one simple interface to a complex system.

Real-Life Examples

  • Remote control for TV setup
  • Food delivery app handling restaurants + payment + delivery
  • "Login with Google" button

Tiny Code Example 

class TV {
  on() {
    console.log("TV ON");
  }
}

class Speaker {
  on() {
    console.log("Speaker ON");
  }
}

class HomeTheaterFacade {
  startMovie() {
    const tv = new TV();
    const speaker = new Speaker();

    tv.on();
    speaker.on();

    console.log("Movie Started 🎬");
  }
}

const theater = new HomeTheaterFacade();
theater.startMovie();
Facade = Simple door to a complicated system 🚪
