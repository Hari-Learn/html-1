🌲 Recursive Descent Parser

A Recursive Descent Parser is a top-down parsing technique where each grammar rule is implemented as a recursive function. It reads tokens one by one and builds a syntax tree according to the language grammar.

How It Works

  1. Read the next token from the input.
  2. Call the function for the current grammar rule.
  3. Recursively parse nested rules.
  4. Return the completed syntax tree.

Pseudo Code

parseExpression() {
    parseTerm();

    while (hasOperator()) {
        parseTerm();
    }
}

Commonly used in compilers, interpreters, programming language parsers, and configuration file parsers because it is simple to implement and easy to understand.
