Distributed Systems

Lamport Clock

Lamport Clock is a logical timestamp algorithm that provides a consistent ordering of events across distributed systems without relying on synchronized physical clocks.

Order distributed events logically

Each process maintains a logical counter that increases whenever an event occurs. During message exchange, timestamps are shared so that every process can determine the correct logical order of events, ensuring consistency in distributed applications.
