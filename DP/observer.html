Observer Design Pattern

Notifies dependents automatically whenever the state changes.

What is Observer Pattern?

The Observer Pattern defines a one-to-many dependency between objects. When one object changes state, all its dependents are notified and updated automatically.

Pattern Flow

  1. 1 Subject maintains a list of observers.
  2. 2 Observers subscribe to the subject.
  3. 3 State changes occur in the subject.
  4. 4 All observers receive automatic updates.

Simple Example 

class Subject {
  observers = [];

  subscribe(observer) {
    this.observers.push(observer);
  }

  notify() {
    this.observers.forEach(o => o.update());
  }
}
