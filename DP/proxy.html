Proxy Design Pattern

Imagine you want to enter a VIP party 🎉

But first, a security guard checks:

  • Do you have a ticket?
  • Are you allowed inside?
  • Is the party full?

The guard controls access to the real party.

Proxy Pattern says:

"Use a helper object to control access to another object."

Simple Definition

A Proxy stands in front of the real object and controls how it is used.

Real-Life Examples

  • Security guard at a building
  • Parental control on apps
  • Loading images only when needed
  • Password-protected websites

Tiny Code Example 

class RealVideo {
  play() {
    console.log("Playing video 🎬");
  }
}

class VideoProxy {
  constructor(age) {
    this.age = age;
    this.video = new RealVideo();
  }

  play() {
    if (this.age >= 18) {
      this.video.play();
    } else {
      console.log("Access Denied 🚫");
    }
  }
}

const user = new VideoProxy(15);
user.play();
