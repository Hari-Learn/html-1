💡 What Is Proxy?

A Proxy is a helper object that stands in front of the real object and decides whether someone can use it.

🔄 How It Works

➜

✅ Why Use It?

  • ✔ Controls access
  • ✔ Improves security
  • ✔ Saves resources
  • ✔ Adds extra logic easily

❌ Downsides

  • ✖ Extra classes to maintain
  • ✖ More complexity
  • ✖ Slight performance overhead

💻 Tiny Code Example

proxy-example.js 
class RealVideo {
  play() {
    console.log("Playing video 🎬");
  }
}

class VideoProxy {
  constructor(age) {
    this.age = age;
    this.video = new RealVideo();
  }

  play() {
    if (this.age >= 18) {
      this.video.play();
    } else {
      console.log("Access Denied 🚫");
    }
  }
}
