CRDT

Conflict-free Replicated Data Types (CRDTs) are specialized data structures that enable multiple replicas to be updated independently while automatically converging to the same final state.

Conflict-free replicated data types

CRDTs eliminate the need for manual conflict resolution by using mathematically designed merge operations. They are widely used in collaborative editing, offline-first applications, and distributed databases where replicas synchronize asynchronously.
