🖥️ Double Buffer Pattern

The Double Buffer pattern uses two buffers: one is displayed on the screen while the other is updated in the background. Once rendering is complete, the buffers are swapped to provide smooth, flicker-free graphics.

How It Works

  1. Draw the next frame in the back buffer.
  2. Keep displaying the front buffer to the user.
  3. When drawing is complete, swap the two buffers.
  4. Repeat for every frame.

Pseudo Code

while (running) {
    draw(backBuffer);
    swap(frontBuffer, backBuffer);
    display(frontBuffer);
}

Commonly used in game engines, graphics rendering, GUI frameworks, and animation systems to eliminate screen flickering and produce smooth visual updates.
