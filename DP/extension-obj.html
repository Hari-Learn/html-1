Extension Object Pattern

Allows adding new functionality to objects dynamically without changing their class.

What is Extension Object?

The Extension Object Pattern separates additional features from the original object. New capabilities can be attached to an object through extension objects without modifying the existing class code.

Example 

class User {
  constructor(name) {
    this.name = name;
    this.extensions = {};
  }

  addExtension(key, extension) {
    this.extensions[key] = extension;
  }

  getExtension(key) {
    return this.extensions[key];
  }
}

const user = new User("John");

user.addExtension("payment", {
  pay() {
    return "Payment completed";
  }
});

console.log(
  user.getExtension("payment").pay()
);
// Payment completed

Key Idea

  • Add new behavior without modifying the original class.
  • Keeps existing code stable and easier to maintain.
  • Supports flexible feature additions at runtime.
  • Useful for plugins, user capabilities, and modular systems.
