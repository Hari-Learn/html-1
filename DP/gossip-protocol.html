Distributed Systems

Gossip Protocol

Gossip Protocol is a decentralized communication mechanism where nodes randomly exchange information with peers, allowing updates to spread efficiently across large distributed systems.

Spread information probabilistically

Instead of broadcasting to every node, each participant periodically shares information with a few random peers. Over multiple rounds, updates propagate rapidly, making gossip protocols scalable, fault-tolerant, and resilient to node failures.
