💡 Definition

The Memento Pattern lets an object save its internal state into a snapshot and restore it later without exposing its private data.

✅ Benefits

  • ✔ Easy rollback
  • ✔ Keeps data encapsulated
  • ✔ Supports undo features
  • ✔ Improves reliability

❌ Drawbacks

  • ✖ Uses extra memory
  • ✖ Many snapshots can become expensive
  • ✖ More classes to manage
