Memento Design Pattern

The Memento pattern is a behavioral design pattern that allows an object to save and restore its previous state without exposing its internal details.

Definition

“Saves and restores object state.”

Common Uses

  • Undo / Redo systems
  • Game checkpoints
  • Text editor history
  • Version snapshots

Benefits

  • Preserves object state
  • Supports rollback functionality
  • Encapsulates internal data
  • Improves maintainability
