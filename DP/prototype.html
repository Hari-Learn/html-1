Prototype Design Pattern

Creates objects by cloning existing ones.

What is Prototype Pattern?

The Prototype Design Pattern is a creational pattern used to create new objects by copying an existing object instead of creating them from scratch.

Benefits

✅ Faster object creation
✅ Reduces repetitive initialization
✅ Improves performance
✅ Useful for complex objects

Java Example 


class Car implements Cloneable {

    String model;

    Car(String model) {
        this.model = model;
    }

    public Object clone() throws CloneNotSupportedException {
        return super.clone();
    }
}

public class Main {
    public static void main(String[] args) throws Exception {

        Car car1 = new Car("BMW");

        Car car2 = (Car) car1.clone();

        System.out.println(car2.model);
    }
}

Real World Example

Photocopying a document is a real-world example of the Prototype Pattern. Instead of creating a new document from scratch, you simply copy the existing one.
