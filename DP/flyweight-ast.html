🌳 Flyweight AST Pattern

The Flyweight AST pattern reduces memory usage by sharing identical nodes in an Abstract Syntax Tree (AST). Instead of creating duplicate objects, common structures are reused.

How It Works

  1. Create an AST node.
  2. Check if an identical node already exists.
  3. If found, reuse the existing node.
  4. Otherwise, create and store a new node.

Pseudo Code

if (cache.contains(node)) {
    return cache.get(node);
}

cache.put(node);
return node;

Commonly used in compilers, interpreters, code analyzers, and language tooling to reduce memory consumption when many identical AST nodes are present.
