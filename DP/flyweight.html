Flyweight Design Pattern

Imagine a game with 10,000 trees 🌳

If every tree stores:

  • Tree image
  • Tree color
  • Tree shape
  • Tree texture

then the game wastes lots of memory 😵

Flyweight Pattern says:

"Reuse shared things instead of creating them again and again."

Easy Example

Instead of storing the same tree image 10,000 times, store it only once and let all trees share it 🌳

Simple Definition

Flyweight saves memory by sharing common data between many objects.

Real-Life Examples

  • Game trees sharing same graphics
  • Emoji icons reused in apps
  • Letters in a text editor

Tiny Code Example 

class TreeType {
  constructor(name) {
    this.name = name;
  }
}

const sharedTree = new TreeType("Green Tree");

const tree1 = sharedTree;
const tree2 = sharedTree;

console.log(tree1 === tree2); // true
Flyweight = Share common stuff to save memory 🧠
