Half-Sync / Half-Async Pattern

The Half-Sync / Half-Async pattern separates an application into two layers:

  • Async Layer – Handles incoming events and requests.
  • Sync Layer – Processes tasks sequentially using worker threads.
Client Request
↓
Async Layer
↓ Queue ↓
Sync Worker Layer
↓
Response

Example: A web server accepts requests asynchronously, places them in a queue, and worker threads process them synchronously.
