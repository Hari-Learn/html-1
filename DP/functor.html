Functor

A Functor is a functional programming pattern that supports mapping a function over wrapped values without changing the structure that contains them.

A Functor allows you to transform a value inside a container while keeping the container itself unchanged.

How It Works

  1. A value is wrapped inside a container.
  2. A function is applied using map().
  3. The value is transformed.
  4. The container remains the same type.

Benefits

  • Encourages functional programming.
  • Keeps transformations predictable.
  • Reduces manual extraction of values.
  • Works well with collections and optional values.

Simple Example

If a Functor contains the value 5, applying a function that doubles the value produces a new Functor containing 10.

Simple Flow

Functor(5)
↓ map(x → x * 2)
Functor(10)
