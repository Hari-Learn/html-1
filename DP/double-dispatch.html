Double Dispatch

Double Dispatch is a design pattern used in object-oriented programming that allows a method to be selected based on the runtime types of two objects involved in a call.

Why Use Double Dispatch?

  • Enables behavior based on multiple object types.
  • Reduces complex conditional logic.
  • Commonly used in collision detection and visitor implementations.

Example 

shape.accept(visitor);

visitor.visit(circle);
visitor.visit(square);

A popular implementation of Double Dispatch is the Visitor Pattern, where the operation executed depends on both the visitor type and the element type being visited.
