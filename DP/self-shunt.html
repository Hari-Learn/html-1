Self-Shunt Testing

Self-Shunt Testing is a unit testing technique in which the test object acts as its own mock. Instead of creating separate mock objects, the test class implements the required interfaces and provides the expected behavior directly, simplifying test setup for certain scenarios.

Key Concept

Test object acts as its own mock. The test class itself fulfills the role of the mock object, making it easier to verify interactions without creating separate mock implementations.

⇄ Acts as Mock
✔ Verify Behavior
