Template Method Pattern

Defines algorithm structure with customizable steps.

What is it?

The Template Method design pattern defines the skeleton of an algorithm in a base class, while allowing subclasses to customize certain steps without changing the overall structure.

Advantages

Common Uses

  • Game development flows
