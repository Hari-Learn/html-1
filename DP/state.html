State Design Pattern

Changes behavior based on internal state.

What is State Pattern?

The State Pattern allows an object to change its behavior when its internal state changes. The object appears to change its class dynamically.

Real World Example

Traffic Light

Changes between Green, Yellow, and Red.

Mobile Phone

Silent, Vibrate, and Ring modes.

Music Player

Playing, Paused, or Stopped states.

Pattern Flow

  1. 1 Object contains multiple states.
  2. 2 Each state defines different behavior.
  3. 3 Internal state changes based on actions.
  4. 4 Object behavior changes automatically.

Simple Example 

class TrafficLight {
  constructor() {
    this.state = "Red";
  }

  change() {
    if (this.state === "Red") {
      this.state = "Green";
    } else if (this.state === "Green") {
      this.state = "Yellow";
    } else {
      this.state = "Red";
    }
  }
}
