🌲 Recursive Descent Parser

Recursive Descent is a parser implementation technique where each grammar rule is represented by a recursive function. It parses source code from the top down, making it simple and easy to understand.

How It Works

  1. Read the next token from the input.
  2. Call the function for the current grammar rule.
  3. Recursively parse nested grammar rules.
  4. Return the completed parse tree or AST.

Pseudo Code

parseExpression() {
    parseTerm();

    while (hasOperator()) {
        parseTerm();
    }
}

Commonly used in compilers, interpreters, scripting languages, and configuration file parsers because of its simplicity and readability.
