CQRS (ELI10)

🧠 What is it?

CQRS means splitting how you read data and how you write data into two separate parts.

❓ Why use it?

Imagine a library:
📖 Reading books = very common
✍️ Writing books = rare

If both happen in the same place, things get slow! So we separate them to make everything faster.

⚙️ How does it work?

👉 One system handles Commands (write)
👉 Another system handles Queries (read)

🧸 Think of it like:
🍳 Kitchen (write food)
🍽️ Dining table (read/eat food)
