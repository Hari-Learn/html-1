DNS Server and DNS Client

What is DNS?

DNS (Domain Name System) is a system that translates human-readable domain names like www.example.com into IP addresses like 192.168.1.1, which computers use to identify each other on a network.

What is a DNS Server?

A DNS Server is a server that stores DNS records and answers DNS queries. When you type a website name in your browser, the DNS server provides the corresponding IP address so your device can connect to the website.

What is a DNS Client?

A DNS Client is usually your computer, mobile phone, or any device that requests information from a DNS server. It sends a query asking, “What is the IP address for this domain name?”

Why Do We Need DNS?

  • Humans can remember names better than numbers
  • DNS makes the internet user-friendly
  • It allows websites to change IP addresses without changing names
  • Improves network organization and management

What is Preferred DNS?

Preferred DNS is the primary DNS server that your device uses first to resolve domain names. If this server is unavailable, the system will try the alternative DNS server.

Example:
Preferred DNS: 8.8.8.8

What is Alternative DNS?

Alternative DNS is a backup DNS server used when the preferred DNS server is not responding. It ensures uninterrupted internet access.

Example:
Alternative DNS: 8.8.4.4

How to Register an FQDN in a DNS Server

  1. Choose a domain name (example.com)
  2. Register the domain through a domain registrar
  3. Access the DNS management panel
  4. Create DNS records (A, AAAA, CNAME, etc.)
  5. Map the FQDN (like www.example.com) to an IP address
  6. Save changes and wait for DNS propagation
