Consistent Hashing

Efficient data distribution in distributed systems

What is Consistent Hashing?

Consistent hashing distributes data across multiple servers such that minimal data is moved when servers are added or removed.

How It Works

  • Both servers and keys are hashed
  • Placed on a circular hash ring
  • Each key is stored on the next server clockwise

Benefits

Scalable

Easy to add or remove servers.

Minimal Rehashing

Only small amount of data moves.

Load Distribution

Helps balance data across nodes.
