Consistent Hashing

Simple & Smart Data Distribution

What is it?
Like putting toys into boxes 🧸📦 without moving everything when boxes change.
How it works
Uses a circle 🎡 where servers & data are placed. Data goes to next server ➡️
Example
Think clock 🕒 → Data at 3 goes to server at 5.
Benefits

🌱 Easy scaling

🚚 Less data movement

⚖️ Balanced load

Bonus
Virtual nodes help make things even smoother 😎
