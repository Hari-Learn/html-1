API Composition Pattern

📌 What is it?

API Composition combines data from multiple microservices into a single response, reducing the number of calls a client must make.

🚀 Why use it?

  • Faster responses
  • Less network calls
  • Simpler client logic

⚠ Challenges

  • Aggregator complexity
  • Handling partial failures
  • Latency from multiple services

🧒 Explain Like I'm 10

Imagine ordering food from a restaurant:

  • One kitchen makes pizza 🍕
  • Another makes drinks 🥤
  • Another makes dessert 🍰

Instead of you going to each kitchen, a waiter collects everything and brings it to you in one tray. 👉 That waiter is API Composition!
