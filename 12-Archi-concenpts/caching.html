Caching

Caching stores frequently accessed data temporarily so it can be retrieved faster without repeatedly querying the database or server, improving performance and speed.

How It Works

  • User requests data
  • System checks cache first
  • If found → return quickly (cache hit)
  • If not → fetch from database (cache miss)

Types of Caching

  • Browser caching
  • Server-side caching
  • Database caching
  • Application caching
