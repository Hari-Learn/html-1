⚡ Caching

🧠 ELI10 Explanation

Imagine you keep your favorite snacks 🍫 on your desk instead of going to the kitchen every time.

Caching is the same idea — store frequently used data nearby so it’s super fast to access.

⚙️ How It Works

  • User asks for data
  • System checks cache first
  • If found → ⚡ instant result (cache hit)
  • If not → fetch from database (cache miss)

📦 Types of Caching

  • Browser caching 🌐
  • Server-side caching 🖥️
  • Database caching 🗄️
  • Application caching ⚙️

👉 Big idea: Don’t fetch the same data again and again — store it once and reuse it for speed!
