API Gateway & BFF Patterns

Understanding modern backend architecture patterns

🌐

API Gateway

A single entry point that manages and routes all client requests to appropriate backend services.

  • Centralized routing
  • Authentication & authorization
  • Rate limiting & caching
  • Logging & monitoring
📱

Backend for Frontend (BFF)

Dedicated backend services designed for specific frontend applications like web, mobile, or desktop.

  • Frontend-specific APIs
  • Optimized data aggregation
  • Reduced over-fetching
  • Improved performance

Key Difference

API Gateway focuses on managing requests across services, while BFF focuses on tailoring responses for specific user interfaces.
