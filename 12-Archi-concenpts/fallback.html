Fallback Pattern

What is it?

A resilience pattern that provides a backup response when a service fails, ensuring smooth user experience.

ELI10

Like carrying an umbrella ☔ — you’re ready when something goes wrong.

Why Important?

  • Better UX
  • High availability
  • Graceful failures

Types

  • Static
  • Cached
  • Default
  • Backup service

Examples

  • Weather app 🌦️
  • E-commerce 🛒
  • Video apps 🎥

Flow

Request → Service → ❌ Fail → Fallback → ✅ Response

Code Example

function getData() {
  try {
    return callService();
  } catch (e) {
    return "Fallback response";
  }
}

Use When

  • External APIs
  • Distributed systems
  • Non-critical features

Avoid When

  • Payments 💳
  • Critical flows
