How Tracert / Traceroute Works

1️⃣ What is Tracert?

A network diagnostic tool that traces the path packets take from source to destination, showing each router (hop) and delay.

tracert google.com

2️⃣ Core Concept

Uses TTL (Time To Live). Each router decreases TTL by 1. When TTL = 0 → Router sends ICMP Time Exceeded.

3️⃣ Protocols Used

  • ICMP → Windows default
  • UDP → Linux default
  • TCP → Optional method

5️⃣ RTT Measurement

RTT = Time Reply Received − Time Sent. 3 probes per hop measure latency.

1 ms | 2 ms | 1 ms

6️⃣ * Meaning

"*" means request timed out:

  • • ICMP blocked
  • • Firewall filtering
  • • Packet loss
