Router on a Stick (ROAS)

Simple Inter-VLAN Routing Explained

What is ROAS?

Router on a Stick (ROAS) is a network configuration where a single physical router interface is used to route traffic between multiple VLANs using subinterfaces.

How It Works

The router connects to a switch using a trunk link. Each VLAN is assigned a subinterface on the router with its own IP address. The router then routes traffic between VLANs.

Why Use ROAS?

ROAS allows devices in different VLANs to communicate without needing an expensive Layer 3 switch, making it ideal for small to medium networks.

Importance

  • Enables inter-VLAN communication
  • Cost-effective routing solution
  • Centralized traffic control
  • Easy to configure and manage

Advantages

  • Low hardware cost
  • Simple network design
  • Works with Layer 2 switches

Limitations

  • Single link can become a bottleneck
  • Lower performance than Layer 3 switches
  • Not ideal for large networks
