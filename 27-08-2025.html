50 Linux Network Commands with Explanations

ifconfig
Displays or configures network interfaces (deprecated in favor of `ip`).
ip addr
Shows IP addresses assigned to all interfaces.
ip link
Displays network interfaces and their state.
ping google.com
Sends ICMP packets to check connectivity to a host.
traceroute google.com
Shows the path taken by packets to reach a host.
netstat -tulnp
Displays listening ports and associated processes.
ss -tulwn
Shows active sockets with faster output than netstat.
nslookup example.com
Queries DNS to get domain name or IP address information.
dig example.com
Another DNS lookup tool with more detailed output than nslookup.
tcpdump -i eth0
Captures network packets on interface eth0.
ssh keygen -t rsa
Use this cmd to generate/create ssh key pair of authentication. Private key stays with you public key goes on the server.
scp file.txt user@remote:/etc
Use this cmd for send the file to remote server in secure way
iptraf -ng
Real time monitoring tool for networking, like who is talking on the network
lsof -i :8080
Shows whos using this port, find out which process using this port in your system
ethtool eno16
Display network interface staus detail info
arp -a
List out the ip with MAC address, nw interface association in a network
hostnamectl status
Shows full system identity and OS info
mtr www.google.com
(My TraceRoot - MTR)- 3rd party tool Real time traceroot+ping
cat /etc/resolv.conf
Display the content of DNS Configuration file on a linux system
iw dev
Display the list of wireless interfaces and their status
ssh-copy-id -i /root/.ssh/id_rsa.pub root@192.168.11.212
Install rsa keys, copied securly to remote user, enable passwordless ssh login
ssh 192.168.11.212
Login to remote server using ssh
nc/ncat/netcat -zv 192.168.11.3
Powerful networking tool, port scanning, port listening, file transfer etc
rsync f1.txt root@192.168.11.1
Delta transfer-transfer only changes,advance tool than cp and scp
nmtui
Network manager text user interface, easy to use network configuration tool
ss -l
Listening sockets
nmcli
Network manager client tool
nmcli device show
Network manager client tool nmcli device nmcli connection show nmcli connection show nmcli device wifi List
nmcli connection show
Network manager client to list the connections list
nmcli device wifi list
Network manager client to list connected device wifi list
