üéØ Scan Types

  • nmap -sS 192.168.1.1 TCP SYN scan (stealth scan)
  • nmap -sT 192.168.1.1 TCP connect scan
  • nmap -sU 192.168.1.1 UDP scan
  • nmap -sA 192.168.1.1 ACK scan-Determines firewall rules, not port state
  • nmap -sN 192.168.1.1 TCP null scan-send packets with no flags used for firewall evasion
  • nmap -sF 192.168.1.1 FIN Scan - Send FIN packets to bypass firewalls(Stealth scan)
  • nmap -sX 192.168.1.1 XMas scan - Send packet with FIN, PSH and URG Flag set
