CCNA Troubleshooting – What You Must Know

Basic Connectivity

  • IP address, subnet mask, gateway errors
  • Interface shutdown or wrong cable
  • Ping & traceroute interpretation

VLAN & Trunking

  • Wrong VLAN on access ports
  • Trunk not forming
  • Native VLAN mismatch

Routing Issues

  • Missing static routes
  • OSPF neighbor problems
  • Routes not in routing table

Wireless Problems

  • SSID or password mismatch
  • Wrong security mode
  • Weak signal or interference

ACL & Security

  • ACL blocking valid traffic
  • Wrong ACL direction
  • Implicit deny issue

Troubleshooting Tools

  • ping, traceroute
  • show ip route
  • show interfaces status
