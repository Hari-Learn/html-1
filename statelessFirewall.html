Firewall Types

🔐

Stateful Firewall

A stateful firewall tracks active connections and makes filtering decisions based on the state of the network traffic and sessions.

  • ✔ Tracks active network sessions
  • ✔ Remembers connection state
  • ✔ Provides stronger security
  • ✔ Used in enterprise networks
  • ✔ Analyzes packet context
⚡

Stateless Firewall

A stateless firewall filters packets individually using rules such as IP address, port number, and protocol without tracking connections.

  • ✔ No session tracking
  • ✔ Each packet checked independently
  • ✔ Faster performance
  • ✔ Lower resource usage
  • ✔ Used in simple filtering systems
