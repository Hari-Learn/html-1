DNS Security & Networking

Split Horizon DNS

Split Horizon DNS provides different DNS responses depending on whether the request comes from an internal network or the public internet.

How It Works

🏢

Internal Network

Employees inside the company network receive private internal IP addresses. 

app.example.com
→ 10.0.0.15
🌍

Public Internet

External users receive public-facing IP addresses for internet access. 

app.example.com
→ 203.0.113.20

Benefits

🔒

Security

Internal systems stay hidden from public users.

⚡

Performance

Internal traffic stays inside the private network.

🌐

Flexibility

Different DNS views for internal and external users.
