SNMP Components (CCNA)

Many CCNA students skip SNMP, but it often appears in exams.

SNMP Manager

The SNMP Manager is the system used to monitor and control network devices. It sends requests to agents and collects network information.

SNMP Agent

The SNMP Agent runs on network devices such as routers and switches. It gathers device information and sends it to the SNMP Manager.

MIB

MIB (Management Information Base) is a database that stores variables describing network device information.

Typical CCNA Exam Question

Question: What database stores SNMP variables?

Answer → MIB
