OSI Model Explanation

Understanding the OSI 7-Layer Model

The OSI (Open Systems Interconnection) model is a conceptual framework used to understand and standardize network communications. It divides networking into seven distinct layers, each with specific functions.

7. Application Layer

Provides network services directly to end users. Examples include HTTP, FTP, DNS, and email protocols.

6. Presentation Layer

Formats and translates data, handles encryption/decryption, and manages compression (e.g., SSL/TLS).

5. Session Layer

Controls connections between devices, manages sessions, including opening, maintaining, and closing them.

4. Transport Layer

Ensures reliable data delivery using protocols like TCP and UDP. Handles flow control and error correction.

3. Network Layer

Responsible for logical addressing (IP addresses) and routing data across networks. Example: IP protocol.

2. Data Link Layer

Handles physical addressing (MAC addresses), error detection, and frames. Divided into MAC and LLC sublayers.

1. Physical Layer

Concerns the physical transmission of raw bitstreams over a medium (cables, electrical signals, wireless frequencies).
